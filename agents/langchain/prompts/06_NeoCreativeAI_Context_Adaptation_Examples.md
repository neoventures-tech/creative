# Exemplos de Adaptações Detalhadas por Contexto

Este arquivo apresenta exemplos de como elementos visuais, personagens, equipamentos, cores e problemas típicos variam conforme o contexto.
Os exemplos servem exclusivamente como referência para adaptação visual.
Eles não representam soluções e não devem ser copiados literalmente.

---

### Equipamentos
Transportadores de correia

Escavadeiras  
Carregadores  

### Cores
Amarelo vibrante para equipamentos

### Personagens
Capacete amarelo ou laranja
